Histórias de Usuário

US-01 — Visualizar produtos

Como usuário, quero visualizar a lista de produtos para escolher um produto para compra.

Critérios de aceite:

Exibir a lista de produtos.
Cada produto deve possuir nome.
Cada produto deve possuir descrição.
Cada produto deve possuir preço.
Cada produto deve possuir imagem.
Cada produto deve possuir botão Add to cart.

US-02 — Ordenar produtos

Como usuário, quero ordenar os produtos para facilitar sua visualização.

Critérios de aceite:

Ordenar por Nome (A → Z).
Ordenar por Nome (Z → A).
Ordenar por Preço (Low to High).
Ordenar por Preço (High to Low).
