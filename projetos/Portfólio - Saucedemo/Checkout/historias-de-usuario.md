Histórias de Usuário  

US-001 – Iniciar o checkout  

Como cliente da loja, quero iniciar o processo de checkout, para prosseguir com a finalização da compra.  

Critérios de aceite  

Ao clicar no botão "Checkout", o sistema deve redirecionar para a página de informações do cliente.  
A página de checkout deve ser carregada corretamente.  

US-002 – Informar os dados do cliente    

Como cliente da loja, quero informar meus dados pessoais, para prosseguir com a compra. 

Critérios de aceite

Os campos First Name, Last Name e Postal Code devem ser obrigatórios.  
Ao preencher todos os campos corretamente e clicar em "Continue", o sistema deve avançar para o resumo da compra.  
Caso algum campo obrigatório não seja preenchido, o sistema deve exibir uma mensagem de erro correspondente.  

US-003 – Visualizar o resumo da compra

Como cliente da loja, quero visualizar o resumo da compra, para confirmar as informações antes de finalizar o pedido.

Critérios de aceite

O sistema deve exibir os produtos selecionados.  
O sistema deve apresentar o subtotal, imposto e valor total da compra.  
Os botões "Finish" e "Cancel" devem estar disponíveis.  

US-004 – Finalizar a compra

Como cliente da loja, quero concluir a compra, para receber a confirmação do pedido.

Critérios de aceite

Ao clicar no botão "Finish", o sistema deve concluir a compra.  
O sistema deve exibir a mensagem de confirmação da compra.  
O botão "Back Home" deve estar disponível.
